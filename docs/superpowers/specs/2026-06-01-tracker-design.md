# Трекер занятий — дизайн

## Что строим

Один файл `index.html`. Личный трекер занятий по КЭП и Олмат для одного пользователя. Данные в `localStorage`. Никакого бэкенда, никакой сборки.

## Архитектура

Один HTML-файл (~700 строк): `<style>` + `<body>` + `<script>`. Vanilla JS.

Данные: массив объектов в `localStorage['tracker-lessons']`:
```js
{ id, type: 'кэп'|'олмат', org, date, time, topic, done: bool }
```

## Блоки

### 1. Фоновый слой
- Несколько радиальных градиентов (`div.blob`), анимированных через `@keyframes` (движение позиций)
- CSS-переменные `--bg-base`, `--blob-a`, `--blob-b` меняются при смене вкладки через JS
- SVG-фильтр `feTurbulence` поверх всего для зерна
- Transition на `:root` переменных — 600ms ease

Цвета:
- Все: `#111118` / blobs `#1f1f2e`, `#16161f`
- КЭП: `#110d00` / blobs `#3a2800`, `#241800`
- Олмат: `#08080f` / blobs `#141430`, `#0a0a22`

### 2. Шапка
- Название «Занятия» (Cormorant Garamond, крупно)
- Счётчик «N на этой неделе» (справа, мелко)
- Три вкладки: Все / КЭП / Олмат
- Скользящий индикатор: абсолютный `div.tab-indicator`, JS обновляет `left` и `width` по `getBoundingClientRect`

### 3. Лента карточек
- CSS Grid / flex-колонка, gap 12px
- Сортировка: ближайшие сверху (по `date + time`)
- Карточка: `backdrop-filter: blur(12px)`, `background: rgba(255,255,255,0.04)`, тонкая граница
- Содержимое: бейдж типа (КЭП/ОЛМАТ), организация, дата+время, тема
- Пройденные: `opacity: 0.35`, тема зачёркнута
- Каскадное появление: `animation: fadeUp 0.4s ease both`, `animation-delay: N * 60ms`
- Клик по карточке → открыть модалку редактирования
- Клик по чекбоксу/галочке → toggle `done`

### 4. Модалка
- `<dialog>` нативный элемент
- Поля: select (КЭП/Олмат), input организация, input date, input time, textarea тема
- Кнопки: «Сохранить» + «Удалить» (только при редактировании, с `confirm()`)
- FAB-кнопка «+» внизу справа → открыть модалку создания

### 5. Пустое состояние
- Показывается когда на активной вкладке 0 занятий
- Текст: «Пока занятий нет — добавь первое» + кнопка «Добавить»
- КЭП изначально пустой → показывает этот экран

## Демо-данные (Олмат)
4 занятия, разные даты (июнь 2026), разные организации:
- Центр «Сириус», разбор задач ВсОШ
- МФТИ, теория чисел
- Олимпиадная школа, комбинаторика
- Онлайн-курс, алгебра

## Шрифты
- Cormorant Garamond (Google Fonts) — заголовки
- Golos Text (Google Fonts) — всё остальное

## Решённые вопросы
- JS: vanilla, без зависимостей
- Цветовая схема: Золото / Индиго (вариант B)
- Демо-данные: да, для Олмат
- Счётчик недели: включён
- Пройденные: не скрываются, приглушаются (toggle)
- Empty state: на русском
